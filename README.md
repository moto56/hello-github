（そんな事や題名とかで騒ぐやついる）

②トラブル、不祥事
兎に角、締結した「契約書」を何度も読む。行間も感じるまで読む。そして意見（コンサルティング）に入る。
・読んで読解して、気になったらコンサル開始　「論点やリスクストーリー」を判りやすく「提案（いくつか提案）」→「承認者の結論をもらう」という作業
③契約書はまずいつもの、「有効・失効・日付・署名押印・保有ルール」から必ず入る。基本の型は忘れない。
